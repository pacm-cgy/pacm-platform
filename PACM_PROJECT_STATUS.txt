슬로건       : 청소년 창업가를 위한 올인원 플랫폼

  - Build    : Vite 8, vite-plugin-pwa (PWA 지원)
  - UI       : 완전 커스텀 CSS (디자인 시스템 v9 — Pure Black × Electric Blue)

  - DB       : Supabase (PostgreSQL + RLS + Realtime)
  - 인증     : Supabase Auth (이메일/비밀번호)
  - 배포     : Vercel (자동 CI/CD)
  - AI 엔진  : 완전 자체 개발 (외부 LLM API 0원)

  - 배경색   : #050505 ~ #181818 (Pure Black 계열)
  - 주요 색상: #3B82F6 (Electric Blue)

[3-1] PACM-AI 메인 엔진 ✅ 완료
──────────────────────────────────────────────────────
파일: api/ai-mentor.js (1030줄) — Vercel Edge Function

  § 1. 한국어 NLP 엔진
    - 한국어 불용어 70개+ 필터
    - 유니그램 + 바이그램 토크나이저
    - TF-IDF 가중치 계산
    - BM25 스코어링 알고리즘 (k1=1.5, b=0.75)

  § 2. 지식 그래프 (개념 관계 맵)
    - 27개 핵심 창업 개념 노드
    - 개념 간 연결 관계 정의
      예: "린캔버스" → [비즈니스모델, MVP, 고객세그먼트, 수익구조, ...]
      예: "투자" → [VC, 시드, 시리즈A, 엔젤, 크라우드펀딩, ...]
    - expandQueryWithGraph(): 쿼리 토큰 → 관련 개념 자동 확장

  § 3. 의도 분류기 v2 (복합 의도 처리)
    - 15개 의도 카테고리:
        lean_canvas, mvp, revenue_model, idea_validation, pitch_deck,
        team_building, market_analysis, funding, government_support,
        startup_basics, marketing, legal_tax, failure_lesson,
        simulation, research_request, greeting, general
    - 복합 의도 처리 (primary + secondary intent)
    - 신뢰도 점수 (confidence score) 반환
    - 컨텍스트 기반 의도 보정 (이전 대화 참조)

  § 4. 자체 리서치 엔진 (핵심 차별점!)
    - selfResearch() 함수: DB 4개 소스 동시 탐색
      ① ai_knowledge 테이블 (BM25 검색)
      ② articles 테이블 (최신 아티클 검색)
      ③ trend_snapshots 테이블 (트렌드 수치 데이터)
      ④ community_posts 테이블 (실제 사용자 경험)
    - 확장 쿼리 토큰 사용 (지식 그래프 확장 후 검색)
    - 다중 소스 결과 병합 및 중복 제거
    - buildResearchSummary(): 검색 결과 → 자연어 합성

  § 5. 시뮬레이션 엔진 (자체 계산)
    - runSimulation() 함수: 5가지 시뮬레이션 타입
      ① revenue: 수익 모델 시뮬레이션
          입력: 사용자수, 전환율, 단가, 성장률
          출력: 월별 수익 예측표, 손익분기점, 12개월 누적
      ② market: 시장 규모 TAM/SAM/SOM 계산
          입력: 전체 인구, 타깃 비율, ARPU
          출력: TAM/SAM/SOM 수치, 시장 점유율 목표
      ③ growth: 성장 시나리오 (보수/기본/공격적 3가지)
          입력: 초기 사용자, 성장률
          출력: 3시나리오별 6개월 예측
      ④ funding: 투자 단계 분석
          입력: 현재 MRR, 팀 규모, 성장률
          출력: 적정 투자 단계, 예상 밸류에이션
      ⑤ lean_canvas: 린 캔버스 자동 평가
          입력: 9개 블록 내용
          출력: 섹션별 점수 + 개선 포인트
    - extractSimParams(): 사용자 메시지에서 숫자 파라미터 자동 파싱

  § 6. 동적 응답 합성기 (고정 템플릿 없음!)
    - synthesizeResponse() 함수
    - buildKnowledgeBlock(): 리서치 결과 → 동적 지식 블록
    - buildTrendBlock(): 트렌드 데이터 → 수치 포함 분석
    - buildContextRef(): 이전 대화 맥락 참조 블록
    - buildSimBlock(): 시뮬레이션 결과 → 포맷된 표
    - buildFollowUp(): 의도별 맞춤 후속 질문 제안
    - 매 응답마다 리서치 결과 + 트렌드 + 맥락을 실시간 조합

  § 7. 리서치 요약 생성기
    - buildResearchSummary(): 4개 소스 합성
    - 지식베이스 인사이트 추출
    - 최신 아티클 요약 포함
    - 트렌드 수치 자동 인용
